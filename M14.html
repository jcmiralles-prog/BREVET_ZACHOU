Thème du jour : Fonctions affines : interprétation et équations
Écris toujours la formule de la fonction avant de calculer. Pour résoudre f(x)=g(x), montre toutes les étapes.
Exercice 1 — Fonctions affines : location de voiture (Brevet 2024 Asie)

Une agence propose 3 formules pour louer une voiture :

Formule A : 0,50 € par km | Formule B : 300 € + 0,25 € par km | Formule C : 900 € kilométrage illimité

La distance Strasbourg-Marseille aller-retour est de 2 × 803 = 1606 km.

Question 1

Exprime en fonction de x (nombre de km) le coût de chaque formule.

💡 Ce qu'on attend : Formule A : pas de forfait fixe. Formule B : forfait + coût/km. Formule C : prix fixe.

✓︎ Correction

Formule A : f_A(x) = 0,50x

Formule B : f_B(x) = 0,25x + 300

Formule C : f_C(x) = 900

📌 Méthode Une fonction affine s'écrit f(x) = ax + b. Ici a = coût par km (pente) et b = forfait fixe (ordonnée à l'origine).
Question 2

Montre que la Formule B coûte 701,50 € pour 1606 km.

💡 Ce qu'on attend : Remplace x par 1606 dans f_B(x). Montre le calcul.

✓︎ Correction

f_B(1606) = 0,25 × 1606 + 300 = 401,50 + 300 = 701,50 € ✓

📌 Méthode Calcul de l'image : on remplace x par la valeur numérique et on effectue les opérations dans l'ordre.
Question 3

Calcule le coût des formules A et C pour 1606 km. Quelle est la formule la plus avantageuse ?

💡 Ce qu'on attend : Calcule f_A(1606) et f_C(1606), puis compare les trois prix.

✓︎ Correction

f_A(1606) = 0,50 × 1606 = 803 €

f_C(1606) = 900 €

Comparaison : Formule B (701,50) < Formule A (803) < Formule C (900)

La Formule B est la plus avantageuse pour ce trajet.

📌 Méthode Comparer des fonctions affines : calculer l'image pour la valeur x donnée et comparer les résultats. Toujours conclure en nommant la meilleure option.
Question 4

Résous l'équation 0,25x + 300 = 0,5x. Interprète ce résultat.

💡 Ce qu'on attend : Regroupe les x d'un côté. Résous. Puis explique ce que signifie ce résultat en termes de location.

✓︎ Correction

0,25x + 300 = 0,5x
300 = 0,5x − 0,25x
300 = 0,25x
x = 300 / 0,25 = 1200 km

Interprétation : pour 1200 km, les formules A et B coûtent le même prix. En dessous de 1200 km, la formule A est moins chère. Au-delà, la formule B est plus avantageuse.

📌 Méthode Résoudre f_A(x) = f_B(x) donne le point d'intersection des deux droites, c'est-à-dire la distance à partir de laquelle une formule devient plus intéressante que l'autre.
