Thème du jour : QCM + Médiane
Le QCM du brevet ne demande pas de justification, mais tu dois connaître le raisonnement. Ici, la correction explique tout ! Pour la médiane, la méthode de classement est obligatoire.
Exercice 1 — QCM (d'après Brevet 2022 et 2023 Métropole)

Clique sur la bonne réponse. La correction apparaît aussitôt.

1. On considère la fonction f définie par f(x) = 3x² − 7. Quelle affirmation est correcte ?
A. 29 est l'image de 2 par f
B. f(3) = 20
C. f est une fonction affine
✓ Bonne réponse ! f(2) = 3×4 − 7 = 12 − 7 = 5. Attends, recalculons : f(2) = 3×2² − 7 = 12 − 7 = 5. Mais "29 est l'image de 2" signifie f(2) = 29, ce qui est faux. En fait la bonne réponse est B car f(3) = 3×9 − 7 = 27 − 7 = 20.
✓ Bonne réponse ! f(3) = 3 × 3² − 7 = 3 × 9 − 7 = 27 − 7 = 20. C'est bien vrai.
A est fausse : f(2) = 3×4 − 7 = 5, pas 29.
C est fausse : f(x) = 3x² − 7 est une fonction du second degré, pas affine (une fonction affine s'écrit ax + b).
✗ À revoir. Une fonction affine s'écrit f(x) = ax + b. Ici f(x) = 3x² − 7 contient un x², ce n'est pas une fonction affine. La bonne réponse est B : f(3) = 3×9 − 7 = 20.
📌 Méthode Pour vérifier une affirmation sur une fonction, on remplace x par la valeur dans la formule et on calcule. Ne pas oublier d'appliquer l'exposant avant de multiplier.
2. On a mesuré les tailles (en m) de 7 élèves : 1,46 ; 1,65 ; 1,6 ; 1,72 ; 1,7 ; 1,67 ; 1,75. Quelle est la médiane ?
A. 1,72
B. 1,67
C. 1,65
✗ À revoir. Il faut d'abord classer les valeurs dans l'ordre croissant : 1,46 ; 1,6 ; 1,65 ; 1,67 ; 1,70 ; 1,72 ; 1,75. Il y a 7 valeurs, donc la médiane est la 4e valeur : 1,67.
✓ Bonne réponse ! On classe dans l'ordre croissant : 1,46 ; 1,6 ; 1,65 ; 1,67 ; 1,70 ; 1,72 ; 1,75. Il y a 7 valeurs. La médiane est la valeur du milieu, c'est-à-dire la 4e valeur : 1,67.
✗ À revoir. Il faut classer les valeurs : 1,46 ; 1,6 ; 1,65 ; 1,67 ; 1,70 ; 1,72 ; 1,75. La médiane est la 4e valeur (il y en a 7), donc c'est 1,67, pas 1,65.
📌 Méthode "médiane" 1) Classer toutes les valeurs en ordre croissant. 2) Si nombre de valeurs impair : médiane = valeur du milieu (position (n+1)/2). 3) Si nombre de valeurs pair : médiane = moyenne des deux valeurs du milieu.
3. Dans le triangle ABC rectangle en A, avec AB = 3, BC = 5, AC = 4. Quelle est la valeur de cos(angle en B) ?
A. 0,8
B. 0,75
C. 0,6
✗ À revoir. cos(B) = côté adjacent / hypoténuse = AB / BC = 3/5 = 0,6. Le côté adjacent à l'angle B est AB (= 3), pas AC.
✗ À revoir. cos(B) = côté adjacent / hypoténuse = AB / BC = 3/5 = 0,6. L'hypoténuse est le côté opposé à l'angle droit (ici BC = 5).
✓ Bonne réponse ! cos(B) = côté adjacent à B / hypoténuse = AB / BC = 3/5 = 0,6. Le côté adjacent à l'angle B est AB = 3. L'hypoténuse est BC = 5 (côté opposé à l'angle droit en A).
📌 Méthode "cosinus" cos(angle) = côté adjacent / hypoténuse. L'hypoténuse est toujours le côté opposé à l'angle droit. Le côté adjacent est le côté qui "touche" l'angle sans être l'hypoténuse.
Exercice 2 — Médiane et analyse de données (d'après Brevet 2022)

On relève les performances au lancer du poids (en mètres) de 13 élèves :

3,41 ; 5,25 ; 5,42 ; 4,30 ; 6,11 ; 4,28 ; 5,15 ; 3,70 ; 6,07 ; 5,82 ; 4,62 ; 4,91 ; 4,01

Question 1

Classe ces valeurs dans l'ordre croissant et détermine la médiane.

💡 Ce qu'on attend : écris toutes les valeurs classées, puis identifie la valeur du milieu. Il y a 13 valeurs, donc la médiane est la 7e.

✅ Correction

Classement croissant : 3,41 ; 3,70 ; 4,01 ; 4,28 ; 4,30 ; 4,62 ; 4,91 ; 5,15 ; 5,25 ; 5,42 ; 5,82 ; 6,07 ; 6,11

Il y a 13 valeurs. La médiane est la valeur de rang (13 + 1) / 2 = 7e valeur.

La 7e valeur est 4,91 m.

📌 Méthode "médiane avec n impair" 1) Classer en ordre croissant. 2) Médiane = valeur de rang (n+1)/2. Ici n = 13 donc rang = 7. La médiane partage la série en deux moitiés : 6 valeurs en-dessous, 6 valeurs au-dessus.
Question 2

Calcule la moyenne de ces performances, arrondie au centième.

💡 Ce qu'on attend : moyenne = somme de toutes les valeurs ÷ nombre de valeurs. Montre la somme et la division.

✅ Correction

Somme = 3,41 + 5,25 + 5,42 + 4,30 + 6,11 + 4,28 + 5,15 + 3,70 + 6,07 + 5,82 + 4,62 + 4,91 + 4,01 = 63,05

Moyenne = 63,05 ÷ 13 ≈ 4,85 m

📌 Méthode "moyenne" Moyenne = (somme de toutes les valeurs) / (nombre de valeurs). Attention : médiane ≠ moyenne. La médiane est la valeur du milieu après classement ; la moyenne est le "résultat équitable" si on partage tout également.
