Exercice 1 — Thalès, Pythagore, homothétie (Brevet 2024 Centres étrangers)

Points E, A, F alignés ; E, B, D alignés ; droites (FD) ∥ (AB).
AE = 4,4 cm ; EB = 3,3 cm ; AB = 5,5 cm ; BD = 6,6 cm.

Question 1

Démontre que le triangle ABE est rectangle.

💡 Ce qu'on attend : Calcule AE² + EB² et compare à AB². Si égaux → réciproque de Pythagore.

✓︎ Correction

AE² = 4,4² = 19,36
EB² = 3,3² = 10,89
AB² = 5,5² = 30,25

AE² + EB² = 19,36 + 10,89 = 30,25 = AB²

D'après la réciproque du théorème de Pythagore, le triangle ABE est rectangle en E.

📌 Méthode Réciproque de Pythagore : calculer les carrés des 3 côtés. Si grand côté² = somme des deux autres², triangle rectangle en face du grand côté.
Question 2

Calcule l'angle ABE arrondi au degré.

💡 Ce qu'on attend : Triangle rectangle en E. cos(ABE) = côté adjacent à B / hypoténuse = EB / AB.

✓︎ Correction

cos(ABE) = EB / AB = 3,3 / 5,5 = 0,6

ABE = arccos(0,6) ≈ 53°

📌 Méthode Dans un triangle rectangle, cos(angle) = adjacent/hypoténuse. Utiliser la touche arccos (cos⁻¹) de la calculatrice.
Question 3

Calcule la longueur FD. (Les droites (FD) et (AB) sont parallèles ; E, A, F alignés ; E, B, D alignés.)

💡 Ce qu'on attend : Thalès dans le triangle EFD avec (AB) ∥ (FD) : EA/EF = EB/ED = AB/FD.

✓︎ Correction

ED = EB + BD = 3,3 + 6,6 = 9,9 cm
EF/EA = ED/EB → EF = EA × ED/EB = 4,4 × 9,9/3,3 = 4,4 × 3 = 13,2 cm

D'après Thalès : AB/FD = EB/ED
5,5/FD = 3,3/9,9 = 1/3
FD = 5,5 × 3 = 16,5 cm

📌 Méthode Thalès : écrire les 3 rapports égaux. Identifier les longueurs connues et calculer l'inconnue. Vérifier le rapport : ED/EB = 9,9/3,3 = 3.
Question 4

Quel est le rapport de l'homothétie de centre E qui transforme le triangle EAB en EFD ?

💡 Ce qu'on attend : Le rapport = longueur image / longueur originale. Utilise les longueurs calculées.

✓︎ Correction

ED/EB = 9,9/3,3 = 3

Le rapport de l'homothétie est 3.

📌 Méthode Rapport d'homothétie = longueur image / longueur de départ. Ici chaque côté de EFD est 3 fois plus grand que le côté correspondant de EAB.
