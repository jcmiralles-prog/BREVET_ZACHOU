Exercice 1 — Programme de calcul : factorisation (Brevet 2023 Polynésie)

Programme : ① Choisir x. ② Calculer x². ③ Multiplier par 2. ④ Ajouter x. ⑤ Soustraire 66. Résultat : 2x² + x − 66.

Question 1

a) Vérifie qu'avec x = 4 le résultat est −30. b) Avec x = −3, le résultat est-il −51 ?

💡 Ce qu'on attend : Applique le programme étape par étape pour chaque valeur.

✓︎ Correction

a) x=4 : 4²=16 ; 2×16=32 ; 32+4=36 ; 36−66 = −30 ✓

b) x=−3 : (−3)²=9 ; 2×9=18 ; 18+(−3)=15 ; 15−66 = −51 ✓

📌 Méthode Attention aux priorités : le carré s'applique à tout le nombre (−3)² = 9, pas −(3²) = −9.
Question 2

On sait que 2x² + x − 66 = (2x − 11)(x + 6). Vérifie cette factorisation en développant.

💡 Ce qu'on attend : Développe (2x − 11)(x + 6) avec la double distributivité.

✓︎ Correction

(2x − 11)(x + 6) = 2x×x + 2x×6 − 11×x − 11×6

= 2x² + 12x − 11x − 66

= 2x² + x − 66 ✓

📌 Méthode Vérifier une factorisation = développer et retrouver l'expression de départ. C'est une vérification rapide et obligatoire.
Question 3

Quels nombres de départ donnent 0 ? Résous (2x − 11)(x + 6) = 0.

💡 Ce qu'on attend : Produit = 0 → l'un des facteurs = 0. Résous chaque équation.

✓︎ Correction

(2x − 11)(x + 6) = 0

2x − 11 = 0 → 2x = 11 → x = 5,5

x + 6 = 0 → x = −6

Les deux nombres de départ sont 5,5 et −6.

📌 Méthode Règle du produit nul : AB = 0 ↔ A = 0 ou B = 0. Résoudre chaque équation séparément.
