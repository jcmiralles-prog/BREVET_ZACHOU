⚡ L'essentiel à retenir
La dictée préparée au brevet : on te donne le texte à l'avance pour que tu puisses le travailler. Le jour J, on te le dicte et tu dois l'écrire sans faute. Les points perdus : fautes d'accord, d'orthographe lexicale, et oublis de majuscules ou de ponctuation. Chaque faute coûte un demi-point ou un point selon le barème.
Ce que le brevet évalue : les accords (sujet-verbe, adjectif, participe passé), l'orthographe des mots courants, et la ponctuation de base.
📏 Les 5 règles les plus importantes pour la dictée
Règle 1 — L'accord sujet-verbe

Le verbe s'accorde toujours avec son sujet, même quand ils sont séparés par des mots.

Ex : "Les élèves de la classe travaillent bien." (sujet = "les élèves", pluriel → -ent)

Piège : "La liste des candidats est longue." (sujet = "la liste", féminin singulier, pas "candidats")

Règle 2 — L'accord de l'adjectif

L'adjectif s'accorde en genre (masculin/féminin) et en nombre (singulier/pluriel) avec le nom qu'il qualifie.

Ex : "une belle journée" (féminin singulier) / "de beaux souvenirs" (masculin pluriel)

Piège : les adjectifs de couleur. "des robes bleues" (accord) mais "des robes bleu clair" (pas d'accord quand deux mots).

Règle 3 — Le participe passé

Avec être : le participe s'accorde avec le sujet. "Elle est partie."

Avec avoir : le participe s'accorde avec le COD seulement si le COD est placé avant le verbe. "La lettre qu'il a écrite." (COD "que" = "la lettre", féminin singulier)

Avec avoir, COD après : pas d'accord. "Il a écrit une lettre."

Règle 4 — Les homophones grammaticaux

Des mots qui se prononcent pareil mais s'écrivent différemment :

  • a / à : "a" = verbe avoir (remplaçable par "avait"). "à" = préposition.
  • on / ont : "ont" = verbe avoir pluriel (remplaçable par "avaient"). "on" = pronom.
  • son / sont : "sont" = verbe être (remplaçable par "étaient"). "son" = déterminant.
  • ou / où : "où" = lieu ou choix avec accent (remplaçable par "ou bien" → si non = accent).
  • ces / ses / c'est / s'est : "c'est" = présentatif. "s'est" = verbe pronominal. "ses" = déterminant pluriel. "ces" = démonstratif.
Règle 5 — Les finales verbales à ne pas confondre
  • -er / -é / -ez : infinitif (marcher) / participe passé (marché) / 2e personne du pluriel (marchez). Astuce : si on peut remplacer par "vendre" → infinitif (-er). Si on peut remplacer par "vendu" → participe (-é).
  • -ai / -ais / -ait : futur 1re pers. (je marcherai) / imparfait ou conditionnel (je marchais / je marcherais).
🎯 Stratégie pour bien préparer la dictée
  • Étape 1 — Lire le texte lentement en entier. Comprendre le sens aide à éviter les erreurs d'accord.
  • Étape 2 — Repérer tous les verbes. Pour chaque verbe, trouver son sujet et vérifier l'accord.
  • Étape 3 — Repérer tous les noms et adjectifs. Vérifier le genre et le nombre de chaque adjectif.
  • Étape 4 — Repérer les participes passés. Avec être ou avoir ? Le COD est-il avant ou après ?
  • Étape 5 — Repérer les homophones. Pour chaque "a", "son", "ou"… vérifier avec le test de substitution.
  • Étape 6 — Apprendre par cœur les mots difficiles. Liste les mots que tu ne connais pas, apprends-les.
  • Étape 7 — Relire au moins deux fois pendant la dictée (une fois pour le sens, une fois pour les accords).
Conseil brevet : Lors de la relecture, ne lis pas mot à mot. Lit phrase par phrase en te posant toujours ces deux questions : "Qui fait l'action ?" (accord sujet-verbe) et "De quoi parle-t-on ?" (accord adjectif/participe).

📚 Mots-clés à maîtriser
Homophone
Mot qui se prononce de la même façon qu'un autre mais qui s'écrit différemment. Ex : "a" et "à", "son" et "sont".
Accord
Modification d'un mot (verbe, adjectif, participe) pour qu'il corresponde en genre et en nombre au mot avec lequel il est lié.
Participe passé
Forme du verbe utilisée pour construire les temps composés (passé composé, plus-que-parfait). Il s'accorde selon des règles précises.
COD
Complément d'objet direct : le mot qui répond à "quoi ?" après le verbe. Il commande l'accord du participe passé avec avoir quand il est placé avant le verbe.
Substitution
Technique pour vérifier un homophone : on remplace le mot par un autre pour voir lequel convient. Ex : "a" → remplacer par "avait" : si ça marche, on écrit "a".

Exercice type brevet
⏱ Durée conseillée : 20 min

Texte de dictée préparée

Ce matin-là, les habitants du village s'étaient levés tôt. Le soleil, qui apparaissait lentement derrière les montagnes enneigées, projetait une lumière dorée sur les toits mouillés. Les enfants, encore ensommeillés, couraient vers l'école en bavardant joyeusement. Leur maîtresse, madame Lefèvre, les avait attendus patiemment devant la grande porte en bois. Elle leur a souri en les voyant arriver.

Question 1 — Préparer la dictée6 pts

Relevez dans ce texte : (a) deux accords sujet-verbe à justifier, (b) un participe passé et expliquez son accord.

✓ Correction

(a) Accords sujet-verbe :

• "les habitants… s'étaient levés" : sujet = "les habitants" (masculin pluriel) → participe "levés" au masculin pluriel (auxiliaire être).

• "les enfants… couraient" : sujet = "les enfants" (pluriel) → verbe "couraient" à la 3e personne du pluriel.

(b) Participe passé :

"attendus" : auxiliaire avoir (avait attendus). COD = "les" (mis pour "les enfants", masculin pluriel, placé avant le verbe) → accord au masculin pluriel : "attendus".

Méthode : Pour les participes avec avoir, pose la question "Quoi ?" ou "Qui ?" après le verbe. Si la réponse est avant le verbe → accord. Si après → pas d'accord.
Rappel : "avait attendus" → "elle avait attendu qui ?" → "les" (= les enfants, masculin pluriel, placé avant) → accord : -us.
Question 2 — Homophones4 pts

Choisissez la bonne orthographe et justifiez chaque choix :

1. Les enfants on / ont couru vers l'école.

2. Elle a / à souri en les voyant.

3. Le soleil brillait ou / où les nuages avaient disparu.

4. Sa / Ça sentait bon la pluie fraîche.

✓ Correction

1. ont — "Les enfants ont couru." On peut remplacer par "avaient" → ça marche → "ont" (verbe avoir).

2. a — "Elle a souri." On peut remplacer par "avait" → ça marche → "a" (verbe avoir). Si c'était une préposition, on ne pourrait pas dire "avait souri" seul.

3. où — "là où les nuages avaient disparu" → le "où" indique un lieu. On ne peut pas le remplacer par "ou bien" → c'est "où" avec accent.

4. ça — "Ça sentait bon" = cela sentait bon → pronom démonstratif. "Sa" est un déterminant possessif (sa maison, son livre).

Méthode des substitutions :
"on/ont" → remplacer par "avaient" : marche = "ont".
"a/à" → remplacer par "avait" : marche = "a".
"ou/où" → remplacer par "ou bien" : marche = "ou" sans accent.
"ça/sa" → remplacer par "cela" : marche = "ça".
Question 3 — Réécriture10 pts

Réécrivez la phrase suivante en mettant les mots soulignés au pluriel. Faites tous les accords nécessaires.

"Le jeune enfant curieux courait vers son professeur qui l'avait appelé."

✓ Correction

"Les jeunes enfants curieux couraient vers leur professeur qui les avait appelés."

Justification des accords :

  • Les → article défini pluriel (le → les)
  • jeunes → adjectif au pluriel (jeune → jeunes)
  • curieux → adjectif déjà en -x au singulier → reste "curieux" au pluriel
  • couraient → verbe au pluriel (courait → couraient)
  • leur → déterminant possessif pluriel (son → leur)
  • les avait appelés → COD "les" (= les enfants, masculin pluriel) placé avant → participe accordé : "appelés"
Méthode de réécriture :
1. Identifie le ou les mots mis au pluriel (ici "enfant" → "enfants").
2. Cherche tous les mots qui s'accordent avec lui : article, adjectifs, verbes.
3. Cherche les déterminants possessifs qui changent (son → leur).
4. Cherche les participes passés : le COD change de genre/nombre ?
Rappel : "curieux" est l'un des rares adjectifs déjà en -x au masculin singulier. Au pluriel, il reste "curieux" (pas de changement). Autres exemples : "heureux", "joyeux", "nerveux".
