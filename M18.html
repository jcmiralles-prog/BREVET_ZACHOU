Thème du jour : Équations du 2ème degré et factorisation
Développe soigneusement, ramène à 0, factorise, puis applique la règle du produit nul.
Exercice 1 — Équation du 2ème degré : factorisation (Brevet 2024 Polynésie)

On considère les fonctions f(x) = (x+2)² − x et g(x) = 7x + 4.

On cherche à résoudre l'équation f(x) = g(x).

Question 1

Calcule f(−4).

💡 Ce qu'on attend : Remplace x par −4 dans f(x) = (x+2)² − x. Montre le calcul.

✓︎ Correction

f(−4) = (−4+2)² − (−4) = (−2)² + 4 = 4 + 4 = 8

📌 Méthode Remplacer x par une valeur : attention aux parenthèses et aux signes. (−4+2) = −2, et (−2)² = 4 (positif !).
Question 2

Détermine un antécédent de 3 par la fonction g.

💡 Ce qu'on attend : On cherche x tel que g(x) = 3. Pose 7x + 4 = 3 et résous.

✓︎ Correction

7x + 4 = 3
7x = 3 − 4
7x = −1
x = −1/7

L'antécédent de 3 par g est −1/7.

📌 Méthode Antécédent de k par g : résoudre g(x) = k. C'est une équation en x à résoudre.
Question 3

Montre que l'équation f(x) = g(x) se ramène à x² − 4x = 0.

💡 Ce qu'on attend : Développe f(x), pose f(x) = g(x) et simplifie. Montre toutes les étapes.

✓︎ Correction

f(x) = (x+2)² − x = x² + 4x + 4 − x = x² + 3x + 4

f(x) = g(x) :
x² + 3x + 4 = 7x + 4
x² + 3x + 4 − 7x − 4 = 0
x² − 4x = 0 ✓

📌 Méthode Pour ramener à une équation = 0 : développer le membre gauche, passer tout à gauche, simplifier. On obtient une équation de la forme ax² + bx = 0 qu'on peut factoriser.
Question 4

Factorise x² − 4x et déduis-en les solutions.

💡 Ce qu'on attend : Mets x en facteur commun. Puis résous le produit = 0.

✓︎ Correction

x² − 4x = x(x − 4)

L'équation x(x−4) = 0 a deux solutions :
x = 0 ou x − 4 = 0
x = 0 ou x = 4

Les solutions de f(x) = g(x) sont x = 0 et x = 4.

📌 Méthode Produit = 0 : un produit est nul si et seulement si l'un de ses facteurs est nul. x(x−4) = 0 → x = 0 ou x = 4. Toujours vérifier : f(0)=(0+2)²−0=4 et g(0)=7×0+4=4 ✓
