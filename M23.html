Exercice 1 — Programme de calcul (Brevet 2024 Centres étrangers)

Programme : ① Choisir un nombre x. ② Calculer (x − 2). ③ Calculer (x + 1). ④ Multiplier les deux résultats.

Question 1

a) Montre qu'avec le nombre 5, le résultat est 18.
b) Calcule le résultat avec −3/2.

💡 Ce qu'on attend : Applique le programme étape par étape. Pour b), garde les fractions jusqu'au bout.

✓︎ Correction

a) x = 5 : (5−2) = 3 ; (5+1) = 6 ; 3 × 6 = 18 ✓

b) x = −3/2 :
(−3/2 − 2) = −3/2 − 4/2 = −7/2
(−3/2 + 1) = −3/2 + 2/2 = −1/2
−7/2 × (−1/2) = 7/4

📌 Méthode Programme de calcul : appliquer chaque étape dans l'ordre. Montrer chaque résultat intermédiaire. Pour les fractions, convertir les entiers en fractions de même dénominateur.
Question 2

Soit g(x) = x² − x − 2. Prouve que (x − 2)(x + 1) = x² − x − 2.

💡 Ce qu'on attend : Développe (x − 2)(x + 1) avec la double distributivité. Montre que tu retrouves x² − x − 2.

✓︎ Correction

(x − 2)(x + 1) = x×x + x×1 − 2×x − 2×1

= x² + x − 2x − 2

= x² − x − 2 ✓

📌 Méthode Double distributivité : chaque terme du 1er × chaque terme du 2e. Regrouper les termes en x.
Question 3

Résous (x − 2)(x + 1) = 0. Déduis-en les antécédents de 0 par g.

💡 Ce qu'on attend : Un produit = 0 si et seulement si l'un des facteurs = 0.

✓︎ Correction

(x − 2)(x + 1) = 0

Soit x − 2 = 0 → x = 2
Soit x + 1 = 0 → x = −1

Les antécédents de 0 par g sont 2 et −1.

📌 Méthode Règle du produit nul : A × B = 0 ⟺ A = 0 ou B = 0. Les solutions sont les antécédents de 0 (valeurs de x qui donnent g(x) = 0).
Question 4

Quels nombres de départ du programme donnent 0 comme résultat ?

💡 Ce qu'on attend : Le programme calcule (x−2)(x+1). Donc le résultat est 0 quand (x−2)(x+1)=0.

✓︎ Correction

Le programme calcule (x−2)(x+1) = g(x).

g(x) = 0 pour x = 2 et x = −1 (question précédente).

Les nombres de départ sont 2 et −1.

📌 Méthode Lien programme ↔ fonction : le programme calcule exactement g(x). Ses sorties nulles correspondent aux antécédents de 0 par g.
