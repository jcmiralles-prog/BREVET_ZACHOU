Exercice 1 — QCM : fonctions et identités (Brevet 2023 Polynésie)

QCM : justifie ton raisonnement pour chaque question (au brevet réel ce n'est pas demandé, mais c'est ce qui te permet de progresser).

Question 1

f(x) = −2x + 3. Quelle est la bonne représentation graphique : croissante ou décroissante ? Quelle est l'ordonnée à l'origine ?

💡 Ce qu'on attend : Le coefficient de x (pente) détermine si la fonction est croissante ou décroissante. L'ordonnée à l'origine = f(0).

✓︎ Correction

f(x) = −2x + 3 : le coefficient de x est −2 (négatif) → la fonction est décroissante.

f(0) = −2×0 + 3 = 3 → l'ordonnée à l'origine est 3.

📌 Méthode Fonction affine f(x) = ax + b : si a > 0 → croissante ; si a < 0 → décroissante. L'ordonnée à l'origine = b (valeur quand x = 0).
Question 2

Développe (3x − 7)².

💡 Ce qu'on attend : Utilise l'identité remarquable (a − b)² = a² − 2ab + b². Ici a = 3x et b = 7.

✓︎ Correction

(3x − 7)² = (3x)² − 2×3x×7 + 7²

= 9x² − 42x + 49

📌 Méthode Identité (a−b)² = a² − 2ab + b². À connaître par cœur. Vérification : développer (3x−7)(3x−7) = 9x²−21x−21x+49 = 9x²−42x+49 ✓
Question 3

On donne le tableau de valeurs de h(x) = −x + 1. Dans un tableur, si la valeur de x est en cellule A1, quelle formule calcule h(x) ?
A) =−A1+1   B) =−A1   C) =−B1+1   D) =A1+1

💡 Ce qu'on attend : La formule doit reproduire −x + 1 en utilisant la cellule A1 pour x.

✓︎ Correction

h(x) = −x + 1, avec x en cellule A1.

La formule est =−A1+1 (réponse A).

B est faux (oublie +1). C est faux (utilise B1 au lieu de A1). D est faux (signe +).

📌 Méthode Tableur : la formule reproduit le calcul mathématique en remplaçant x par la référence de cellule. Vérifier chaque option.
