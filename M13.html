Thème du jour : Calcul littéral : factorisation et programmes
Montre chaque étape de développement. Pour factoriser, identifie le facteur commun ou l'identité remarquable.
Exercice 1 — Factorisation (Brevet 2024 Asie)

On considère l'expression : 4x² − 9

Question 1

Identifie la forme de cette expression et factorise-la.

💡 Ce qu'on attend : Reconnais une identité remarquable : a² − b² = (a − b)(a + b). Ici a = 2x et b = 3.

✓︎ Correction

4x² − 9 = (2x)² − 3²

C'est une différence de deux carrés : a² − b² = (a−b)(a+b)

Avec a = 2x et b = 3 :

4x² − 9 = (2x − 3)(2x + 3)

📌 Méthode Identité remarquable : a² − b² = (a−b)(a+b). Pour la reconnaître : est-ce que chaque terme est un carré parfait ? Ici 4x² = (2x)² et 9 = 3². Vérification : (2x−3)(2x+3) = 4x²+6x−6x−9 = 4x²−9 ✓
Question 2

Vérifie ta factorisation en développant (2x − 3)(2x + 3).

💡 Ce qu'on attend : Utilise la double distributivité et montre que tu retrouves 4x² − 9.

✓︎ Correction

(2x − 3)(2x + 3) = 2x×2x + 2x×3 − 3×2x − 3×3

= 4x² + 6x − 6x − 9

= 4x² − 9 ✓

📌 Méthode Pour vérifier une factorisation, on redéveloppe et on doit retrouver l'expression de départ. C'est une vérification rapide et efficace.
Exercice 2 — Calcul littéral : programme de calcul (Brevet 2021 Métropole)

Programme de calcul :

① Choisir un nombre. ② Ajouter 2. ③ Prendre le carré du résultat. ④ Soustraire le carré du nombre de départ.

Question 1

Vérifie qu'avec le nombre 2 comme départ, on obtient 12.

💡 Ce qu'on attend : Applique chaque étape du programme au nombre 2. Montre toutes les étapes.

✓︎ Correction

① On choisit 2.
② 2 + 2 = 4
③ 4² = 16
④ 16 − 2² = 16 − 4 = 12 ✓

📌 Méthode Appliquer un programme : suivre chaque étape dans l'ordre. Ne jamais sauter d'étape. Montrer le résultat intermédiaire à chaque étape.
Question 2

Si on choisit x comme nombre de départ, montre que le résultat est (x+2)² − x² = 4x + 4.

💡 Ce qu'on attend : Développe (x+2)² = x² + 4x + 4, puis soustrait x². Montre le développement.

✓︎ Correction

(x+2)² − x² = x² + 4x + 4 − x²

Les x² se simplifient :

= 4x + 4 ✓

📌 Méthode Développement de (x+2)² = (x+2)(x+2) = x²+2x+2x+4 = x²+4x+4. Identité à connaître : (a+b)² = a²+2ab+b².
Question 3

Si on choisit un nombre entier, le résultat est-il toujours un multiple de 4 ? Justifie.

💡 Ce qu'on attend : Le résultat est 4x + 4 = 4(x + 1). Est-ce un multiple de 4 ? Pourquoi ?

✓︎ Correction

Le résultat du programme est 4x + 4 = 4(x + 1).

4(x+1) est le produit de 4 par (x+1). Quel que soit l'entier x, 4(x+1) est toujours divisible par 4.

Donc oui, le résultat est toujours un multiple de 4.

📌 Méthode Pour montrer qu'une expression est un multiple de 4, on la factorise pour faire apparaître le facteur 4. Si l'expression s'écrit 4×(quelque chose), c'est un multiple de 4.
